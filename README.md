# Getex

All-in-one platform for:
- 🎌 Watching Anime
- 📈 Boosting Social Media (via Telegram bot)
- ⌨️ Creating Keyboard Styles

## Tech Stack
- React 18 + TypeScript
- Vite
- Tailwind CSS
- Hosted on Vercel

## Getting Started

```bash
npm install
npm run dev
```
